# 成员管理概述
<Note>
**本模块提供实验室成员的权限管理能力，实现创建人对实验室成员的完整控制，包括成员邀请、信息查看、成员移除及实验室删除等操作。**
</Note>

## 权限划分⭐️
* **实验室创建人**：系统自动赋予最高权限
* **普通成员**：仅具备基础参与权限

## 核心功能🍉
### 1. **成员邀请**
    * 仅创建人可操作实验室的【邀请成员】按钮
    * 点击后生成唯一时效性邀请链接(24小时有效)
    * 支持链接复制及多渠道分享（邮件/即时通讯等）
    * 被邀请人通过链接可自动加入实验室

### 2. **成员信息**
    + 创建人可查看：实时成员数量统计、成员详细列表（含用户名、邮箱、手机号、角色等信息）
    + 普通成员无成员列表查看权限

### 3. **成员移除**
    + 创建人可通过成员列表的【删除】操作踢出成员
    + 被移除成员立即失去实验室访问权限
    + 被移除成员再次加入需重新获取邀请

### 4. **删除实验室**
    + 仅创建人可操作【删除实验室】功能
    + 执行删除前需二次确认
    + 删除后所有成员权限立即失效
<Warn>
**删除实验室后数据将永久消失⚠️**
</Warn>

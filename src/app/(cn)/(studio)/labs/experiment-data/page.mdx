# 3D模型概述
<Note>
**Uni-Lab-OS的3D模型系统采用标准的ROS2生态系统,具有良好的扩展性和兼容性。主要的3D模型文件以STL格式存储,支持精确的几何表示和物理仿真。**
</Note>

## 核心功能模块
### 3D设备建模系统
* **机械臂建模**: 支持多关节机械臂的精确3D建模
* **XYZ定位平台**: 三轴运动平台的完整几何定义
* **样品处理设备**: 实验室专用设备的3D几何建模
### 物理仿真引擎
* **关节运动仿真**: 支持旋转关节和直线关节的物理约束
* **碰撞检测**: 设备间碰撞检测和避障
* **运动控制**: ROS2控制接口集成
### 设备集成管理
* **设备注册**: 统一的设备配置管理
* **MoveIt集成**: 运动规划和路径优化


## 用户场景
* **预演复杂流程，规避设备碰撞与误操作**
* **优化实验室空间布局与工作动线**
* **提供零风险虚拟操作培训**

## 核心功能
* **3D仿真加速方案设计**
* **图形化编程简化运动控制**
* **电气拓扑图一键调试**

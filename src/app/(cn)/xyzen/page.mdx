# Xyzen

Xyzen 是 ScienceOL 矩阵中的 AI 担当。

<Note>
1. 免费开源
2. 无需整站部署，快速集成于现有网站
3. 自演化 Agent 框架
</Note>

## Features

Xyzen 最大的特点是通过 NPM Package 方式一键集成于你的现有网站中，来快速为你的服务添加 AI 能力。
**不同于其他的 LLM Client 开源方案需要整站部署**，
Xyzen 专注于核心的 LLM 业务逻辑。可基于你的网站风格进行高度定制化。

### 前端集成

在你的 React 项目中使用 Xyzen：

```bash
yarn add @sciol/xyzen@latest
```

基本用法：

```tsx
import { Xyzen, useXyzen } from '@sciol/xyzen'

function App() {
  const [isXyzenOpen, panelWidth] = useXyzen() 
  return (
    <main>
        <div style={{
            marginRight: isXyzenOpen ? panelWidth : 0,
        }}>
            {children}
        </div>
        <Xyzen />
    </main>
  )
}
```

### 2. 一键创建 MCP 服务器，并与各 AI 提供商集成

Xyzen 支持通过一键部署 MCP（Model Context Protocol）服务器，
轻松连接主流 AI 服务商（如 OpenAI、阿里云、百度、智谱等），实现模型统一管理和智能路由。
你可以根据业务需求，灵活切换和组合不同 AI 能力，提升系统的智能化水平和扩展性。

# UI 使用及设计规范
本章节介绍 Xyzen 的 UI 设计规范和原则，帮助开发者和设计师创建一致且用户友好的界面。

## @sciol/ui 组件库

Xyzen 使用 [@sciol/ui](https://www.npmjs.com/package/@sciol/ui) 
作为主要的前端组件库。
该组件库基于 shadcn 和 vercel 构建，提供了一套高度可定制且易于使用的 UI 组件。
